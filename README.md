# Netflix-Data-Analyist
Project Overview
This project provides an in-depth analysis of Netflix's vast content library, uncovering meaningful insights about movies, TV shows, genres, production trends, and viewer preferences. By leveraging data analytics, we aim to understand the streaming giant's content strategy, regional distribution, and evolving entertainment landscape.
 Objectives:
Explore Netflix's content catalog structure and composition

Identify trends in content production over the years

Analyze genre popularity and distribution

Investigate regional content availability

Study the ratio of movies vs. TV shows and their growth patterns

Analyze seasonal release patterns and content ratings

Uncover hidden patterns in cast, directors, and content duration

Dataset Information
Source: [e.g., Kaggle Netflix Dataset / Official Netflix Data / Web Scraped]

File: netflix_titles.csv

Records: 8,000+ titles

Features:

show_id – Unique identifier for each title

type – Movie or TV Show

title – Name of the content
